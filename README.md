Sugar Bash Helper
=================

### Clone SugarCRM CodeSniffer

* 
  ```
  git clone git@github.com:svnvcristea/sugarBash.git
  ```

### SetUp config file

Copy config.def.yml to config.yml

  ```bash
cd ./sugaBash
cp -r ./config_def.yml ./config.yml

  ```

Check your config.yml file and adjust accordingly

### Run the helper

  ```bash
sudo bash helper.sh

  ```
